# RTW

## Install
* Clone from: https://github.com/vandijkstef/real-time-web.git
* CD into real-time-web
* npm install
* To start the application run command: npm run start
* [Have fun on http://localhost:3000](http://localhost:3000)

## Dependencies
* Cookieparser
* WS 
* EJS
* Express
* Express-Session
* Http-errors
* Morgan
* Node-emoji
* Node-sass-Middleware
* Session-file-store

## Features
* Select nation flag
* Chat
* Native WS
